# 기초 문법

## 조건문 if
    if 조건문: 
        수행할 문장
    else:
        수행할 문장

if문은 다음와 같은 문법을 가지며 조건식이 참일 경우 if문 안에 있는 문장을 수행하고
거짓일 경우 수행하지 않고 넘어가거나 다음 조건식으로 넘긴다.

## 반복문 for
``` Python
for 변수 in (반복 가능 객체)
```

반복 가능 객체에는 여러가지 종류가 있다.
1. range() - 숫자 범위

range() 의 괄호 안에 숫자를 어떻게 넣냐에 따라 변수에 대입되는 값이 달라진다.
range(시작값, 종료값, 간격)으로 값을 넣을 수 있다.

2. 리스트 

["원소1", "원소2"]의 값이 각각 들어간다.

3. 튜플
4. 문자열
5. 딕셔너리
6. 집합

## 반복문 while
``` Python 
while 조건식:
    수행할 문장
```

만약 조건식이 참이라면 수행할 문장을 실행한다.

## 함수
``` Python
def 함수이름(매개변수1, 매개변수2, ...):
    실행할 코드
    return 반환값
```
