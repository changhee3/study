# 변수
> 변수란 =기호를 기준으로 오른쪽 값을 왼쪽 값에 대입하여 대신 사용하는 기호이다. 객체를 대신 표현하는 이름표라고 생각해도 좋다.
```Python
a = "apple"       /// 문자열 자료형 변수 할당 ///
```

## 변수를 선언하는 규칙
1. 영문자, 숫자, 언더스코어(_)만 사용할 수 있다.
2. 숫자로 변수명을 시작할 수 없다.
3. 예약어는 사용할 수 없다(예 : print, if, and 등)
4. 변수명은 대소문자를 구분한다.

### 변수의 특징
1. 변수의 타입을 미리 선언하지 않는다.
2. 값을 할당하는 순간부터 사용할 수 있다.
3. 값을 가변적으로 변화시킬 수 있다.
4. 함수 안에서 변수를 할당했다면 지역변수라고 부르고 해당하는 함수 안에서만 변수를 사용할 수 있다.
5. 여러 이름이 하나의 객체를 공유할 수 있다.
```Python
a = "apple"       /// 문자열 자료형 변수 할당 ///
b = a             /// b와 a는 같은 값을 갖고 다른 이름을 갖는다. ///
```
