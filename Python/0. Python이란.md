## Python이란?

- 인터프리터 언어이며 소스코드를 한 줄 한 줄 내려가며 읽어 즉시 번역하고 실행하는 번역 프로그램이다.
- 주로 인공지능, 데이터 분석, 웹 개발 분야에 유용하게 사용된다.
- 구글, 인스타그램, 넷플릭스, 아마존 등의 많은 기업에서 파이썬을 주로 사용한다.
- 하지만 Python은 매우 빠른 속도를 요구로 하는 프로그램이나 하드웨어에 직접적인 영향이 필요한 프로그램에는 적합하지 않다.
- 추가적으로 모바일 프로그래밍을 하는데도 어려움이 있다.

### Python의 특징

1. Python은 사람들이 이해하기 쉽게 만든 고급 언어이다.
2. 무료 오픈 소스 코드이다.
   - 오픈 소스란 누구나 자유롭게 사용하고 수정할 수 있도록 소스 코드를 공개한 소프트웨어이다.
  
3. 코드 작성이 간결해서 유지보수에 유용하다.
4. 시스템 유틸리티 제작에 유용하다.
   - 시스템 유틸리티란 컴퓨터 자채를 다루며 작업을 자동화 해주는 도구이다.
   - Python이 윈도우나 맥 등의 운영체제와 대화하는 기능이 내장되어 있어 파일을 이동, 수정, 삭제 등의 파일작업이 수월하다.

5. 사물 인터넷 분야에도 활용도가 높다.
